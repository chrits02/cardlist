Card List README!

By Phawee Toraparppisan

นี่เป็นครั้งแรกของผม ในการทดสอบอัพโหลดไฟล์ไว้บน github และเปิดใช้เป็นเว็บไซต์ น่าเสียดายมากที่มันไม่สามารถรัน PHP ได้ และผมก็ลืมไปว่าไฟล์นี้มีการเชื่อมต่อไปยัง database 5555+ ดังนั้นผมจึงใส่หน้านี้เข้ามาแทน

วิธีการใช้ไฟล์นี้ เราต้องโหลดลงมาด้วย terminal : git clone https://github.com/chrits02/cardlist.git

ติดตั้ง sql เข้าไปแก้รหัส database ใน dbs.php และก็จะทำให้สามารถใช้งานไฟล์นี้ได้ อาจรันบน localhost หรือ server ก็ได้ แต่มันไม่สามารถรัน php บน github ได้

ดังนั้น หน้าเว็บนี้เลยอยากให้เป็นหน้าที่บันทึกสมองในการใช้ github ไว้ตรงนี้

การใช้ GitHub ให้เราสร้าง github ขึ้นมาอันนึง จากนั้นเอาลิ้งก์จาก Quick setup ก็อปมา และเข้าไปที่ terminal ไปที่ folder ที่ต้องการวางไฟล์ จากนั้น
ใช้คำสั่ง git clone [url ที่ก็อปไว้]
รันคำสั่งนี้ ระบบจะสร้าง folder ไว้ในหน้าที่เราเข้าอยู่
ให้เรา cd เข้าไปที่ folder นั้น ลากไฟล์เว็บที่ต้องการใส่ github ลงไปใน folder จากนั้น ใช้คำสั่งบน terminal ในการ push file เข้าไปบนเว็บ Github โดยการพิมพ์

git add -A

git commit -m "ข้อความ comment version ที่แก้ไข"

git push -u origin master

แค่นี้ไฟล์ทั้งหมด ก็จะเข้าไปอยู่ใน เว็บ github ที่เราทำเอาไว้แล้ว หากต้องการ publish ให้เข้าไปที่หน้าเว็บ github -> setting เลื่อนลงมาที่เมนู GitHub Page เลือก source เป็น master branch หน้าเว็บจะรีเฟรช แล้วเลื่อนลงมาดูอีกครั้ง เราจะเจอลิ้งก์ที่ github สร้างไว้ให้ และทำให้เราสามารถเข้าหน้าเว็บนั้นได้เลย
